# 7
```
#include <stdio.h>
int main(void)
{
int i;
int fib[5] = {1, 1};   /* 数组初始化 */
for ( i = 2; i <5; i++ )
fib[i] = fib[i - 1] + fib[i - 2];
for ( i = 0; i <5; i++ )
printf ( "%2d", fib[i] );
return 0;
}
```
上面程序的执行结果是：
`1 1 2 3 5`
